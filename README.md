# ST Silly Achievements

Расширение для [SillyTavern](https://github.com/SillyTavern/SillyTavern), которое добавляет систему ачивок в стиле Steam прямо в ваши чаты. ИИ сам решает, когда наградить вас за яркий момент в ролеплее - неожиданный поворот сюжета, удачную реплику, героический поступок или эпический фейл.

## Как это работает

Расширение инжектит в контекст промпт, который объясняет ИИ формат выдачи ачивок. Когда ИИ решает, что произошло что-то достойное, он добавляет в конец ответа специальный маркер:

```
[ACHIEVEMENT: 💀 | Без права на ошибку | Прошёл сцену один на один с боссом и не дрогнул | epic]
```

Расширение парсит этот маркер, скрывает его из чата и показывает красивое всплывающее уведомление. Все ачивки сохраняются в метаданных чата и доступны через плавающую кнопку.

## Возможности

**Система редкости** - четыре уровня: common, rare, epic, legendary. Каждый уровень имеет своё визуальное оформление и свечение.

**Steam-like тосты** - всплывающие уведомления при получении ачивки. Настраиваются: угол экрана, масштаб, количество строк заголовка и описания, острые/скруглённые углы.

**Тематические пресеты** - 7 встроенных цветовых тем (Soft, Pink, Lavender, Mint, Peach, Night, Blackout) + возможность создавать свои с настройкой 4 цветов (primary, secondary, accent, text).

**Пресеты промптов** - встроенные промпты на русском и английском. Можно создавать, редактировать, сохранять и переключать свои пресеты. Отдельные пресеты для основного промпта и негативного промпта (отправляется во время кулдауна).

**Кулдаун** - настраиваемый интервал между ачивками (в сообщениях). Во время кулдауна вместо основного промпта инжектится негативный, который запрещает ИИ выдавать награды.

**Локальная валидация** - расширение дополнительно проверяет кулдаун на стороне клиента и фильтрует точные дубликаты (оба параметра отключаемы).

**Панель ачивок** - плавающая кнопка (draggable) с счётчиком, по клику открывается панель со списком всех ачивок текущего чата, статистикой по редкостям и информацией о кулдауне. Ачивки можно удалять по одной или все сразу.

**Гибкий инжект** - настраиваемая позиция (до/после промпта, на глубину), роль (system/user/assistant) и частота вставки промпта.

**Отладка** - встроенные инструменты для ручной выдачи ачивок с произвольными параметрами.

**Локализация** - полная поддержка русского и английского интерфейса (определяется автоматически по локали SillyTavern).

## Установка

1. Откройте SillyTavern
2. Перейдите в **Extensions** -> **Install Extension**
3. Вставьте ссылку на репозиторий:
   ```
   https://github.com/niemandswasser/ST-silly-achievements
   ```
4. Нажмите **Install** и перезагрузите страницу

Или клонируйте вручную в папку расширений:

```bash
cd [путь до SillyTavern]/data/default-user/extensions/third-party/
git clone https://github.com/niemandswasser/ST-silly-achievements
```

## Настройки

После установки расширение появится в панели расширений SillyTavern. Настройки разделены на секции:

**Основные** - включение/выключение, кулдаун, локальная фильтрация, дедупликация.

**Инжект промпта** - позиция, глубина, роль, частота вставки, пресеты основного и негативного промптов с полнотекстовым редактором.

**Внешний вид тостов** - включение уведомлений, свечение редкости, угол экрана, масштаб, количество строк, острые углы, тематические пресеты с цветовой настройкой.

**Отладка** - ручная выдача и удаление ачивок, открытие панели.

## Формат маркера

```
[ACHIEVEMENT: эмодзи | Название (до 96 символов) | Описание (до 220 символов) | редкость]
```

Редкости: `common`, `rare`, `epic`, `legendary`

## Совместимость

Работает с любыми моделями, которые способны следовать инструкциям и выводить структурированный маркер. Протестировано с Claude и Gemini.
